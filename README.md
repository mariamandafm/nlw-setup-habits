# nlw-setup-habits
Projeto feito no evento Next Level Week da Rocketseat

## Descrição
O projeto é um aplicação web e mobile, além do backend, que auxilia no acompanhamento de hábitos.
## Tecnologias utilizadas
* Backend:
  * Node.js
  * Fastify
  * Prisma 
* Frontend (web):
  * React.js
* Frontend (mobile):
  * React Native
  * Expo
